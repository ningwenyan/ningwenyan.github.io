Github加速访问和下载

导读

前几天,我们讲解了如何自建一个Hexo博客,在搭建的过程中,就感受到了Github 的访问速度非常慢,而且git clone项目半天下载不下来了.今天就总结几个明显提高速度的方法.

Github提升访问速度

由于不可预知的原因,Github的DNS被污染,所有这里使用修改本机hosts文件的方法,修正其DNS地址.

1.获取Github地址

访问https://www.ipaddress.com/查询以下链接的DNS地址

github.com
assets-cdn.github.com
github.global.ssl.fastly.net

并打开命令行手动ping哪个地址对自己的延迟最小,然后记录下来(注意根据自己的实际情况来)

2.更新地址

windows :打开C:\Windows\System32\drivers\etc\hosts文件,并添加上面的地址和ip对应.

ubuntu:/etc/hosts文件,添加并修改

3.刷新缓存

windows:打开命令行cmd工具,并输入ipcofnig /flushdns

ubuntu:打开terminal,并输入service network restart
