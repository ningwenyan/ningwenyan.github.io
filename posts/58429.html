进程的创建

1.引入进程

在python中多线程其实并不是真正的多线程,如果要充分的使用多核CPU的资源,大部分情况下需要使用多进程.python中提供了多进程包multiprocessing .

2.调动linux系统的fork 函数

linux 系统中提供了一个fork 函数,它非常特殊,普通函数调用,调用一次返回一次,这个函数调用会返回2次,操作系统自动把当前进程(父进程)复制一份(子进程).然后分别在父进程和子进程中返回.

  • os.fork 对子进程永远返回为0,而父进程返回子进程的PID
  • 一个父进程可以fork 出很多子进程
  • 子进程可以通过getpid()拿到自己的PID .通过getppid() 拿到父进程的PID
# fork 子进程
import os

print('Process {} start'.format(os.getpid()))
pid = os.fork()

if pid == 0:
 print('I am chilid Process {}, and the main Process is {}'.
       format(os.getpid(), os.getppid()))
else:
 print("Main Process {}".format(os.getpid()))

print("This is main process and child process")

1.fork 全局变量不能修改

子进程会完全复制一份父进程的内容,他们之间的变量是相互隔离的

a = 1

ret = os.fork()

if ret == 0:
 a += 1
 print(a)
else:
 a += 2
 print(a)

结果:

3
2

3.python内置模块multiprocessing

在multiprocessing 模块中,使用multiprocessing.Process 来创建一个新的进程对象

常用的属性和方法有:

Process方法
Process.start()启动进程,每个进程最多只能调用一次
Process.run()进程的执行逻辑定义在run中,可以在子类中重写方法
Process.join([timeout])阻塞当前进程,直到没有调用join 方法的那个进程执行完毕,再继续执行当前进程
Process.is_alive()判断进程时候存活
Process.terminate()终结一个进程,当调用这个函数的时候,运行逻辑中的exit,finally 代码段将不会执行.而且这个进程的子进程不会被终结而成为孤儿进程
属性
name字符串,进程的名字,可以通过赋值语句修改
pidpid值
authkey获取/设置进程的授权密码,当初始化多进程时,使用os.urandom为主进程分配一个随机字符串,当创建一个Process对象时,它将继承其父进程的秘钥.可以使用authkey设置为其他字符串来改变,它也可以用来获取秘钥
daemon布尔值,为True 时为守护进程.他必须在start()之前设置,否则会引发RuntimeError
exitcode返回进程退出的代码.进程运行时值为None ,如果为-N ,表示被信号N结束
Process函数参数
target表示调用的可调用对象
args表示调用对象的位置参数元组
kwargs表示调用对象的位置参数字典
name别名,进程的名字

1.为进程命名

如果需要在子进程中知道父进程的名字,需要显示声明

name = multiprocessing.current_process().name
name = multiprocessing.current_process().name
import multiprocessing
import time

def foo(my_time):
 # 如果子进程需要知道父进程的名字,需要显示声明
 name = multiprocessing.current_process()
 # multiprocessing.current_process().name
 print('Start {}'.format(name))
 time.sleep(my_time)
 print('Exiting {}'.format(name))

if __name__ == '__main__':
 """主进程"""
 # 生成一个自定义名字的子进程
 process_with_name = multiprocessing.Process(target=foo, args=(3,), name='foo_process')
 process_with_name.start()
 # 生成一个带有默认名字的子进程
 process_with_default_name = multiprocessing.Process(target=foo, args=(3,))
 process_with_default_name.start()

运行结果:

start <Process(foo_process, started)>
start <Process(Process-2, started)>
Exiting <Process(foo_process, started)>
Exiting <Process(Process-2, started)>

可以看到子进程不命名,系统会给个默认的名称.

4.创建子进程

1.创建函数并将其作为单个子进程

import multiprocessing,datetime,time,os

def worker(num):
 n = 5
 while n > 0:
     print('Now is {}'.format(datetime.datetime.now()))
     print(os.getpid(), os.getppid())
     time.sleep(num)
     n -= 1
if __name__ == '__main__':
 proc = multiprocessing.Process(target=worker, args=(3,))
 proc.start()
 # 获取进程授权密码
 print('proc授权密码' ,proc.authkey)
 # 设置授权密码,必须编码为ASCII
 proc.authkey = b'123.qwe'
 # 再次获取授权密码
 print('proc授权密码' ,proc.authkey)
 # 显示进程名称
 print('proc.name:',proc.name)
 proc.name = 'test'
 print('proc.name:', proc.name)
 print('proc是否存活', proc.is_alive())
 print('进程PID',proc.pid)

2.创建函数将其作为多个子进程

import multiprocessing,time,os

def worker_1(interval):
 print('start worker_1')
 print('执行的进程PID:{},父进程PID:{}'.format(os.getpid(), os.getppid()))
 time.sleep(interval)
 print('end worker_1')

def worker_2(interval):
 print('start worker_2')
 print('执行的进程PID:{},父进程PID:{}'.format(os.getpid(), os.getppid()))
 time.sleep(interval)
 print('end worker_2')

def worker_3(interval):
 print('start worker_3')
 print('执行的进程PID:{},父进程PID:{}'.format(os.getpid(), os.getppid()))
 time.sleep(interval)
 print('end worker_3')

if __name__ == '__main__':
 print('主进程 Start,PID:{}'.format(os.getpid()))

 for i in [worker_1, worker_2, worker_3]:
     proc = multiprocessing.Process(target=i, args=(3,))
     proc.start()
     print("The number of cpu is {}".format(multiprocessing.cpu_count()))

 for p in multiprocessing.active_children():
     print('child p.name:{}, p.id:{}'.format(p.name, p.pid))

 print('主进程 End')

3.定义子进程的子类

# 定义进程的一个子类,重写run方法

import multiprocessing,time,random,os

class MyProcess(multiprocessing.Process):
 """重写run方法"""
 def __init__(self):
     super(MyProcess, self).__init__()

 def run(self):
     # self.pid 是 Process类的一个属性值
     print('子进程PID:{},开始时间:{}'.format(self.pid, time.ctime()))
     for i in range(5):
         print('Hello')
         time.sleep(random.random())
     print('子进程:{}:结束'.format(self.pid))

if __name__ == '__main__':
 """主进程"""
 print('主进程PID:{},开始时间:{}'.format(os.getpid(), time.ctime()))
 for i in range(5):
     proc = MyProcess()
     proc.start()
 print('主进程结束')

这相当与重写了Process类的run 方法.

4.带有join()方法的子进程

子进程中调用join() 方法,则父进程必须等待子进程执行完毕后在执行

import os,multiprocessing,time,datetime

def worker_1(my_time):
 info = '子进程开始执行,PID:{},name:{},执行时间是:{},父进程PID:{}'.format(
     os.getpid(), multiprocessing.current_process().name, datetime.datetime.now(), os.getppid())
 print(info)
 time.sleep(my_time)
 print('子进程 {} 退出'.format(multiprocessing.current_process().name))

def worker_2(my_time):
 info = '子进程开始执行,PID:{},name:{},执行时间是:{},父进程PID:{}'.format(
     os.getpid(), multiprocessing.current_process().name, datetime.datetime.now(), os.getppid())
 print(info)
 time.sleep(my_time)
 print('子进程 {} 退出'.format(multiprocessing.current_process().name))


if __name__ == "__main__":
 """主进程"""
 print('主进程开始')
 for i in [worker_1, worker_2]:
     if i == worker_1:
         proc1 = multiprocessing.Process(target=i, args=(5,))
         proc1.start()
         # join 表示主进程要等待子进程执行完毕后再执行
         proc1.join()
     else:
         proc2 = multiprocessing.Process(target=i, args=(2,))
         proc2.start()
         proc2.join()
 print("主进程结束")

执行结果:

子进程开始执行,PID:21180,name:Process-1,执行时间是:2019-11-02 18:30:53.030472,父进程PID:21179
子进程 Process-1 退出
子进程开始执行,PID:21183,name:Process-2,执行时间是:2019-11-02 18:30:58.041658,父进程PID:21179
子进程 Process-2 退出
主进程结束
